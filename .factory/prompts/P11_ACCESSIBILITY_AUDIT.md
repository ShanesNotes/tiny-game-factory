# P11 — Accessibility / Readability Audit

ROLE: Accessibility critic.

TASK:
Audit:
- keyboard/mouse/controller assumptions
- color/contrast/readability
- tutorial clarity
- error/restart paths
- motion/flash risk
- text size
- cognitive load
- child-friendly / audience-specific constraints when relevant

OUTPUT:
`reviews/ACCESSIBILITY.md` with pass/fail and minimum fixes.
