# P04 — First Playable Slice

ROLE: Implementer for one branch.

TASK:
Build the thinnest playable slice that exercises the core loop end to end.

MUST HAVE:
- one complete loop
- terminal state or clear restart/continue
- programmer art only
- deterministic RNG seed
- debug overlay/log
- random bot
- heuristic/skilled bot or scripted bot
- ≥60s smoke run
- unit/sim tests for core logic
- playtest_report.json
- capture or replay

FORBIDDEN:
- content sprawl
- high-fidelity art
- multiplayer backend
- account/auth/persistence beyond local state
- engine migration without ADR
- new dependency without justification

STOP at fun gate. Do not polish.
