# P12 — IP / Originality Audit

ROLE: IP/originality auditor.

TASK:
Identify borrowed names, logos, characters, maps, ability names, icons, copyrighted visual identity, or too-close genre clone details.

RULE:
Genre grammar is allowed. Specific IP expression is not.

OUTPUT:
`reviews/IP_ORIGINALITY.md` with:
- risk list
- required renames/redesigns
- pass/fail
